Changelogs

v4.1 — April 8, 2026

  • Emoji Hub & Custom Text: You can now upload local images right into the Emoji Hub to use directly within App Titles and Descriptions!
  • Extreme Performance Boost: Massive memory savings and rendering optimizations for complex app grids using hardware acceleration and visibility observers.

v4.0 — April 7, 2026 • The Mega Quality-of-Life Update

  • Changelogs, Help & Honorable Mentions: The old boring "Help" button has been completely overhauled with a premium hub. Changelogs live at the top, Help tips are in a collapsible dropdown, and there's a special golden Honorable Mentions section at the bottom.
  • Dev Mode Notice: The bug button now shows a clear dialog about the upcoming external app feature, keeping the UI clean and informative without opening the full dev panel.
  • Dual-Layer Data Persistence: Your apps now save to both localStorage and sessionStorage simultaneously. If localStorage is ever wiped, the session backup kicks in automatically and restores everything — no more losing your apps!
  • Update Backup File: The Sync modal now has a dedicated button to pick a .txt backup from your files and update it with fresh portal data.
  • Collapsible Help Sections: All the feature tips are now organized into smooth accordion dropdowns so the dialog stays clean and easy to navigate.
  • Code Studio Stability: Fixed edge cases where multiline HTML content could lose newlines, and generally improved the editor's save/close flow stability.

v3.5 — The Fun Edition & Quality-of-Life Update

  • App & Category Duplication: Clone any app or entire folder in one click — no need to rebuild from scratch. Duplicates appear right next to the original.
  • Format Painter: Copy an app's full visual style (colors, background, glow, opacity) and paint it onto any other app. Supports bulk painting when multiple apps are selected.
  • Custom Favicons: Set a custom icon for your Portal's browser tab. Supports emoji, image URLs, and uploaded image files. The icon travels with Cloaked Tab windows too.
  • Element Inspector: Activate live text editing directly on the portal — click any visible text to edit it in-place without opening a modal.
  • Multi-Select & Bulk Management: Hold Shift to enter Selection Mode. Tap multiple apps, then use the + button to bulk-edit, move to folders, or delete all at once. The Add button glows to signal an active selection.
  • Fredoka One Typography: Switched the entire portal UI to Fredoka One for a playful, premium feel throughout dialogs, buttons, and menus.
  • Micro-Animations: Added smooth squish and spring effects on cards, buttons, and dialog transitions for a snappier, more responsive feel.
  • Improved Dialogs: All modals now have a clean top-right X close button. Long app names, URLs, and descriptions horizontally scroll so nothing gets cut off.

Help?

Emoji Hub & Rich Text

Go to Settings > Experimental Features > Emoji Hub. Upload local images or add URLs. Once added, type the keyword (e.g. :dog:) in any App Name or Description to render the custom emoji!

Advanced App Customization

Use the Extra Extra Customization menu to upload local background images, set true glassmorphism opacity, and independently color your icons and text.

Security & Access

The admin access link provides a hidden bypass. Use Fast Boot in settings to skip the login screen on your local device.

Category Folders

Categories act as sub-portals. Click the folder icon in the App Creator to make one. You can drag apps onto the Back button to move them up one level.

Code Studio

Embed your own HTML apps. Use the Dev Sidebar to edit Live CSS, App State, or raw Source HTML with instant previews.

Cloud Sync (Local)

Export your entire ecosystem as a single code block or a .txt file to transfer between devices effortlessly.

Quick Tip

If a site shows a blank screen, it's likely blocking iframes. Switch that app to Cloaked Tab mode in the creator settings.
